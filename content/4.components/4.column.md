## Usage
Add the component to your email template. Include styles where needed.

```vue
<script setup>
import { EColumn, ESection } from 'vue-email'
</script>

<template>
  <ESection>
    <EColumn>A</EColumn>
    <EColumn>B</EColumn>
    <EColumn>C</EColumn>
  </ESection>
</template>
```
