## Usage
Add the component to your email template. Include styles where needed.

```vue
<script setup>
import { EHead } from 'vue-email'
</script>

<template>
  <EHead>
    <title>Email title</title>
    <meta name="description" content="Description">
  </EHead>
</template>
```
